# Support

If you need help using or contributing to this project, use these channels:

## 1) GitHub Discussions (recommended)

Use Discussions for:

- setup questions
- architecture/approach questions
- usage guidance
- open-ended design conversations

## 2) GitHub Issues

Use Issues for:

- confirmed bugs
- actionable feature requests
- documentation defects

Please include reproducible steps, expected behavior, and actual behavior.

## 3) Security issues

Do not report security vulnerabilities publicly.

Use the private vulnerability reporting flow described in [SECURITY.md](./SECURITY.md).

## Response expectations

This is an open-source project and maintainers respond as time permits.
Please be respectful and patient.
